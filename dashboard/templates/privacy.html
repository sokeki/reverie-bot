🌙 Reverie — Privacy Policy

Last updated: 2026

What we collect

  • Discord account info: your Discord user ID, username, and avatar, used to identify you in the points/shop system and dashboard. We use Discord's own login (OAuth) for the dashboard, we never see your Discord password.
  • Points/shop activity: your points balance, purchase history, and equipped items.
  • Riot ID and match statistics: for players tracked in the Valorant/TFT features, we store their Riot ID, PUUID, and match statistics (rank, RR, K/D, headshot %, etc.) retrieved via the official Riot Games API, so we can post updates and stat comparisons.
  • Optional: linked Riot session (daily shop feature). If you choose to use /linkriot, we store an encrypted session token so we can show your personal daily shop when you run /dailyshop. This is entirely opt-in. Your Riot password is never seen or stored at any point, only Riot's own login page ever sees it. You can remove this at any time with /unlinkriot, which deletes the stored session immediately.

How we use it

Solely to provide the bot's features: posting updates, displaying stats, running the shop, and fetching your daily shop when requested. We don't use this data for advertising, and we don't sell or share it with third parties, other than the API calls to Riot Games' own servers necessary to retrieve the data itself.

Where it's stored

Data is stored in a private database we control. Sensitive session tokens (from the optional daily-shop feature) are encrypted at rest.

Retention & deletion

Points/shop and match-tracking data is kept for as long as you're tracked in the bot. You can request full deletion of your data at any time by contacting a server admin. Linked Riot sessions can be removed instantly and by yourself with /unlinkriot.

Third-party services

We use the official Riot Games API for match/rank data, and Discord's API for login and bot functionality. Each is governed by its own privacy policy and terms.

Changes

This policy may be updated as the bot's features change. Material changes will be reflected on this page.

Contact

For privacy questions or data deletion requests, contact the server admins through Discord.
